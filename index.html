Screeps Browser Extensions

Collection of userscripts to enhance your Screeps experience. Click on any script to install it with your userscript manager (TamperMonkey, ViolentMonkey, etc.).

  • Alliance Overlay
    Install alliance-overlay.user.js
  • Birthday Viewer
    Install birthday-viewer.user.js
  • Console Shard Color
    Install console-shard-color.user.js
  • Diplomacy Overlay
    Install diplomacy-overlay.user.js
  • Force Alpha Map
    Install force-alpha-map.user.js
  • Gui Extender
    Install gui-extender.user.js
  • Improved Console History
    Install improved-console-history.user.js
  • Migrate Screeps Room
    Install migrate-screeps-room.user.js
  • Room Claim Assistant
    Install room-claim-assistant.user.js
  • Visible Room Tracker
    Install visible-room-tracker.user.js

These userscripts require screeps-browser-core.js to function.
